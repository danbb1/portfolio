# eBay AI Email Generation Agent

## Overview

Developed an AI-powered system using LangChain and LangGraph to generate marketing email content directly from briefs, streamlining content creation for internal teams.

## Key Features & Functionality

Generates text and images via multiple tool calls, producing distinct variations of content based on user preferences.

Implements conversational editing, allowing iterative refinement of content with messages and changes saved across turns.

Automatically validates outputs and retries failed generations, ensuring high-quality content.

## Impact

Reduced manual content creation time for marketing teams.

Provides a flexible, interactive workflow that integrates AI generation with human feedback.

Forms a foundation for further experimentation with AI-driven content tools, including integrations with Figma for direct design-to-content workflows.
